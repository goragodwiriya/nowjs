Paragraphs

นี่คือย่อหน้าที่เกิดจาก tag <p>....</p> คุณสามารถใส่เนื้อหาต่างๆได้ภายใต้ tag นี้

จัดตัวอักษรชิดขวาด้วย <p class=right>...</p>

จัดตัวอักษรกึ่งกลางด้วย <p class=center>...</p>

จัดตัวอักษรชิดซ้ายด้วย <p class=left>...</p>

Code/Blockquote

ใช้สำหรับระบุส่วนที่เป็นโค้ด โดยอยู่ภายใต้ tag <code>....</code>
ใช้สำหรับการอ้างอิงไปยังคำพูด <blockquote>...<blockquote>

Tag <em> used for emphasizing important messages

Tag <strong> and <b> and <dt> display as bold

Tag <i> and <dfn> display as italic

Tag <ins> and <u> display as underline

Tag <del> display as strikethrough

Tag <acronym> and <abbr>

Tag <sub> and <sup> display as subscript and superscript

Tag <small> and <big> for adjusting the font size
