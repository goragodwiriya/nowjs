Editor (Basic Profile)

Welcome to Rich Text Editor!

This is a powerful and flexible WYSIWYG editor built with vanilla JavaScript.

Features:

  • Text formatting (bold, italic, underline, strikethrough)
  • Headings (H1-H6)
  • Lists (ordered and unordered)
  • Text alignment
  • Links and images
  • Tables with context menu
  • Undo/Redo history
  • Keyboard shortcuts
